네, 일반 사용자도 2단계 인증을 활용할 수 있도록 구현하겠습니다. 현재는 설정만 가능하고 실제 로그인 시에는 체크하지 않고 있습니다. 로그인 시에도 2단계 인증을 거치도록 기능을 확장하겠습니다.

**구현 계획:**

1.  **데이터베이스 스키마 수정**: `Session` 모델에 `twoFactorVerified` (인증 완료 여부) 필드를 추가합니다.
2.  **타입 정의 업데이트**: `next-auth` 타입 정의에 2FA 관련 필드(`twoFactorEnabled`, `twoFactorVerified`)를 추가합니다.
3.  **인증 설정 업데이트**: `src/lib/auth/index.ts`에서 세션 콜백을 수정하여 2FA 정보를 세션에 포함시킵니다.
4.  **2FA 인증 페이지 구현**: 로그인 후 2FA가 필요한 경우 이동할 `/verify-2fa` 페이지를 생성합니다.
5.  **2FA 검증 로직 구현**: 입력한 코드를 검증하고 세션의 `twoFactorVerified` 상태를 업데이트하는 Server Action을 구현합니다.
6.  **접근 제어 강화**: 메인 레이아웃(`(main)/layout.tsx`)과 관리자 레이아웃(`admin/layout.tsx`)에서 2FA가 활성화된 사용자는 인증 완료 여부를 확인하고, 미완료 시 인증 페이지로 리다이렉트하도록 수정합니다.

이 작업을 통해 관리자뿐만 아니라 2단계 인증을 설정한 모든 사용자가 로그인 시 추가 인증을 거치게 되어 계정 보안이 강화됩니다.
