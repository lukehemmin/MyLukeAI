2단계 인증(2FA) 기능을 구현하기 위한 계획은 다음과 같습니다.

1.  **패키지 설치**: TOTP 생성 및 검증을 위한 `otplib`와 QR 코드 생성을 위한 `qrcode` 라이브러리를 설치합니다.
2.  **데이터베이스 스키마 변경**: `User` 모델에 2단계 인증 관련 필드(`twoFactorSecret`, `twoFactorEnabled`)를 추가합니다.
3.  **백엔드 로직 구현**:
    *   2FA 시크릿 생성, 활성화, 비활성화, 검증을 처리할 Server Action을 구현합니다.
    *   관리자 권한 확인 미들웨어(`src/lib/auth/admin-middleware.ts`)를 수정하여, 관리자가 2FA를 활성화하지 않았을 경우 `/?openSettings=account&focus=2fa`로 리다이렉트하도록 설정합니다.
4.  **프론트엔드 구현**:
    *   **설정 모달 (`SettingsModal.tsx`)**: 계정 탭에 2FA 설정 섹션을 추가합니다. (QR 코드 표시, 인증 번호 입력, 활성화/비활성화 기능)
    *   **사이드바/레이아웃 (`Sidebar.tsx`)**: URL 쿼리 파라미터(`openSettings`, `focus`)를 감지하여 자동으로 설정 모달을 열고 2FA 섹션으로 스크롤/포커스하는 로직을 추가합니다.

이 계획에 따라 순차적으로 구현을 진행하겠습니다.
